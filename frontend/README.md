# Pipeline Dashboard Frontend

React + Tailwind CSS frontend for the Pipeline Monitoring Dashboard.

## Installation

```bash
cd frontend

# Install dependencies
npm install
```

## Running

```bash
npm run dev   # Runs on http://localhost:3001
```

## Project Structure

```
frontend/
├── src/
│   ├── pages/         # Dashboard, PipelineDetail
│   ├── components/    # Reusable components
│   ├── services/      # API client
│   └── types/         # TypeScript types
```
